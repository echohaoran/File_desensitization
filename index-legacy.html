PRODUCT / v0.1 PROTOTYPE

多步骤脱敏系统

面向企业内部投资人员与私募基金管理员的本地优先型敏感数据脱敏工具。上传文件后系统自动识别敏感信息， 人工复核后生成脱敏文件与字段映射表；凭借映射表可完整还原原始文件。 每个浏览器持有独立用户标识，数据互不交叉。

01 DESENSITIZE

脱敏工作流

上传文本或图片文件，系统自动检测敏感字段并在右侧预览，人工增删脱敏区域后生成脱敏文件与映射表。

  • 文本 / CSV / JSON 正则检测
  • 图片 Canvas 框选遮盖
  • 映射表本地持久化
进入工作流
02 RESTORE

还原工作流

上传此前生成的脱敏文件与对应映射表 JSON，校验用户归属后按映射关系还原原始文件并下载。

  • 映射表用户归属校验
  • 占位符逆向替换
  • 图片原图重建
进入工作流

用户数据隔离：当前浏览器已分配独立用户标识（见页头 USER 标签）。 所有映射表均以该标识为键存入 localStorage，不同浏览器 / 用户之间互不可见。生产环境将改由后端按 uploads/{user_id}/ 目录树与数据库 user_id 外键强制隔离。
